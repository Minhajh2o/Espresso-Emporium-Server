## üöÄ Getting Started

### Prerequisites
- **Node.js** (v16 or higher) - [Download](https://nodejs.org/)
- **npm** (comes with Node.js)
- **MongoDB Atlas Account** - [Sign Up Free](https://www.mongodb.com/cloud/atlas)
- **Git** - For cloning the repository

### Installation

#### 1. Clone the Repository
```bash
git clone https://github.com/Minhajh2o/Espresso-Emporium-Server.git
cd Espresso-Emporium-Server
```

#### 2. Install Dependencies
```bash
npm install
```

#### 3. Configure Environment Variables
Create a `.env` file in the root directory:

```env
DB_USER=your_mongodb_username
DB_PASSWORD=your_mongodb_password
PORT=3000
```

#### 4. Start the Server
```bash
npm start
```

The server will start on `http://localhost:3000`

## üîå API Endpoints

### Base URL
```
http://localhost:3000
```

### Endpoints Overview

| Method | Endpoint | Description | Request Body |
|--------|----------|-------------|--------------|
| GET | `/` | Server health check | None |
| GET | `/coffees` | Get all coffees | None |
| GET | `/coffees/:id` | Get coffee by ID | None |
| POST | `/coffees` | Add new coffee | Coffee object |
| PUT | `/coffees/:id` | Update coffee | Coffee object |
| DELETE | `/coffees/:id` | Delete coffee | None |


### Detailed API Documentation

#### Get All Coffees
```http
GET /coffees
```

#### Get Single Coffee
```http
GET /coffees/:id
```

#### Add New Coffee
```http
POST /coffees
Content-Type: application/json
```
Request body: `{ name, quantity, supplier, taste, category, details, photo, price }`

#### Update Coffee
```http
PUT /coffees/:id
Content-Type: application/json
```

#### Delete Coffee
```http
DELETE /coffees/:id
```

## üíæ Database Schema

### Collection: `coffees`
**Database**: `espressoEmporium`

```javascript
{
  _id: ObjectId,          // Auto-generated by MongoDB
  name: String,           // Coffee name
  quantity: String,       // Available quantity
  supplier: String,       // Supplier name
  taste: String,          // Taste profile
  category: String,       // Coffee category
  details: String,        // Detailed description
```
